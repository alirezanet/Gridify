Gridify

A modern dynamic LINQ library for .NET

Get Started Introduction

Simple and Easy to Use

Gridify provides a powerful string-based dynamic LINQ query language that's simple and easy to use.

Exceptional Performance

Gridify has been designed with performance in mind and can outperform other dynamic LINQ libraries.

Compatibility

Gridify can be used in any .NET Core or .NET Framework project, making it highly compatible. It can also be used alongside Entity Framework and other ORMs. Gridify.Elasticsearch extension provides and ability to build Elasticsearch DSL queries from Gridify filters.
