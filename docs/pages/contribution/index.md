## Documentation

Gridify documentation is powered by vuePress,
All the markdown source files are placed in `docs` directory.

by running the following commands, you can check the documentation site locally:

:::: code-group

```shell [yarn]
yarn install
yarn dev
```

```shell [npm]
npm install
npm run dev
```

::::
