# Getting Started

In order to use Gridify with Elasticsearch it's necessary to install Gridify.Elasticsearchopen in new window.

# Installation

# Package Manager

Install-Package Gridify.Elasticsearch -Version 2.14.1

# .NET CLI

dotnet add package Gridify.Elasticsearch --version 2.14.1

# Namespace

After installing the package, you can use the Gridify namespace to access the package classes and static Extension methods.

using Gridify.Elasticsearch;
